# Data Science Practicals

This repository contains practical exercises and notebooks for introductory data science topics. Below is an overview of each file:

## descriptive_statistics.ipynb
This notebook covers the basics of descriptive statistics, including measures of central tendency (mean, median, mode), measures of dispersion (variance, standard deviation), and data visualization techniques. It includes practical examples and exercises.

## practical_sesssion2.ipynb
This notebook contains exercises and examples from the second practical session. Topics may include data manipulation, basic data analysis, and introductory Python programming for data science.

## practice_questions.ipynb
This notebook provides practice questions to reinforce concepts learned in class. It includes a variety of data science problems and their solutions, focusing on statistics, Python, and data analysis.

## tested.csv
This CSV file is used as a sample dataset for analysis in the notebooks. It contains data relevant to the exercises and examples provided.
